User Stories from Product Backlog that we will address in this sprint:
a) As a user, I should be able to load a previous state of the shell that was 
   created before.
b) As a user, I should be able to save the state of the shell to be used for 
   later times.
c) As a user, I should be able to use cd, popd, pushd, save and load states 
   without any bugs
d) As a user, I should be able to move or copy 
   files within directories to different locations.
e) As a user, I should be able to send the output of a function into a 
   file, and if the file does not exist, create the file with the output in it.
f) As a user, I should be able to use mkdir, tree, find without any bugs
g) As a user, I should have access to documentation of all the commands available
h) As a user, I should be able to direct the output of any command to a file if needed
