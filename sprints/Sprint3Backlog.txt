User Stories from Product Backlog that we will address in this sprint:
a) As a user, I should be able to change directories without any problems
b) As a user i should be able to use the pushd, popd and stack without any 
   problems
c) As a user, I should be able to create directories in a given location,
   or create a directory in the current location if none is given.
d) As a user, I want to be able to see the current working directory with 
   its full path.
e) As a user, I should be able to print a string into the shell, or input
   the string into a given file by deleting all content in file then adding
   the inputted string. If the file doesn't exist, I should be able to create
   the file with the given name.
f) As a user, I should be able to print a string into the shell, or input the
   string into a given file by adding onto the end of the file. If the file
   doesn't exist, I should be able to create the file with the given name.
g) As a user, I should be able to display the contents of a given file with the 
   contents of other files (if given).
h) As a user, I want to be able to see the contents in the current directory, or 
   see the contents of a specific directory. If my input is invalid, I should see 
   an error message.
