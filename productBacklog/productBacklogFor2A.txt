1. As a user, I need to be able to
input commands and see the results of my inputs.            3 days

2. As a user, I should be able to create directories
in a given location, or create a directory in the current
location if none is given.                                  4 days

3. As a user, I should be able to change the current
directory I am in to a different directory.                 3 days

4. As a user, I want to be able to see the contents
in the current directory, or see the contents of a
specific directory. If my input is invalid, I should
see an error message.                                       4 days

5. As a user, I want to be able to see the current
working directory with its full path.                       3 days

6. As a user, I need to be able to push my current
directory onto a stack, then change my new current
directory to a given inputted directory. The pushed
directory should be saved into a stack so that it can
be returned at any time. I should also be able to
push any number of directories into stack.                  6 days

7. As a user, I should be able to pop directories from
the stack, and make the returned directory the current
working directory. If my stack does not have any
directories, I should see an error message.                 6 days

8. As a user, I should be able to see my recent
commands I have previously used and know the order
I have used them in.                                        3 days

9. As a user, I should be able to display the
contents of a given file along with the contents
of other files (if given).                                  4 days

10. As a user, I should be able to print a string
into the shell, or input the string into a given file
by deleting all the content in file then adding the
inputted string. If the file doesn't exist, I
should be able to create the file with the given name.      6 days

11. As a user, I should be able to print a string into
the shell, or input the string into a given file by adding
onto the end of the file. If the file doesn't exist, I
should be able to create the file with the given name.      6 days

12. As a user, I should be able to see the documentation
for every command.                                          3 days

13. As a user, I should be able to exit the program.        3 days
